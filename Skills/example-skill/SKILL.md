# Skill: Process Inbox Note

**When to use**: A new note has landed in `Inbox/` and needs to be triaged
into a permanent note.

**Steps**:
1. Read the inbox note in full.
2. Decide if it's a fleeting thought, a reference, or a permanent note idea.
3. If permanent: create a new note in `Notes/` using `Meta/Templates/note.md`,
   give it a clear atomic title, and write the idea in your own words.
4. Link the new note to at least one related existing note.
5. Delete the original file from `Inbox/`.

**Depends on**: `Meta/Templates/note.md`
