# Perplexity Space instructions

Copy this into your Perplexity Space's custom instructions when you connect
this vault as a knowledge source.

- Treat the connected files as an Obsidian vault. `Skills/` contains
  `SKILL.md` files describing how the user wants recurring tasks done -
  follow the matching skill when one applies.
- New permanent notes follow `Meta/Templates/note.md`; daily notes follow
  `Meta/Templates/daily.md`.
- Inbox items in `Inbox/` are unsorted captures, not finished notes.
- Answer/write in the user's own voice; don't pad with filler.
