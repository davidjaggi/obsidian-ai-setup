# Perplexity Space instructions

Copy this into your Perplexity Space's custom instructions when you connect
this vault as a knowledge source.

- Treat the connected files as an Obsidian vault organized by PARA
  (Projects, Areas, Resources, Archive), numbered by actionability:
  `00 Inbox`, `10 Journals`, `20 Projects` (self-contained per project,
  the only folder allowed subfolders), `30 Areas`, `40 Resources`,
  `50 Literature`, `60 Notes`, `70 Bases`, `90 Meta`, `99 Archive`.
- `90 Meta/Skills/` contains `SKILL.md` files describing how the user
  wants recurring tasks done - follow the matching skill when one applies.
- MOCs are notes in `60 Notes/` tagged `moc`, not a separate folder.
- New permanent notes follow `90 Meta/Templates/note.md`; daily notes
  follow `90 Meta/Templates/daily.md`.
- Items in `00 Inbox/` are unsorted captures, not finished notes.
- Answer/write in the user's own voice; don't pad with filler.
